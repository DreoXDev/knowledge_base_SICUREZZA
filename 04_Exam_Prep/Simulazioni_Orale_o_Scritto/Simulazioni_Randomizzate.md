# Simulazioni Randomizzate

## Metodo

Scegli una domanda per ogni blocco, rispondi a voce o per iscritto, poi correggi con [[Griglia_Valutazione]].

## Blocco crittografia

- Simmetrica vs asimmetrica.
- Firma digitale con message digest.
- Hash e collision resistance.
- DES, Triple DES, AES.

## Blocco sistemi operativi

- ACL e mediazione completa.
- DAC vs MAC.
- Bell-LaPadula con compartimenti.
- Challenge/response e reflection attack.

## Blocco reti

- Packet filter e IP spoofing.
- Packet filter vs application gateway.
- TLS struttura pacchetto.
- IPsec transport vs tunnel.

## Blocco applicazioni

- XSS.
- SQL injection.
- Buffer overflow.
- Heartbleed o null pointer dereference.
