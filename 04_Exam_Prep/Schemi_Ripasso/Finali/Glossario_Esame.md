# Glossario Esame

| Termine | Definizione breve | Collegamento |
|---|---|---|
| Safety | Prevenzione di danni da malfunzionamento. | [[Safety_vs_Security]] |
| Security | Protezione da attacchi intenzionali. | [[Safety_vs_Security]] |
| Confidenzialita' | Accesso alle informazioni solo da parte di autorizzati. | [[CIA_Triad_DAD]] |
| Integrita' | Protezione da modifica non autorizzata. | [[CIA_Triad_DAD]] |
| Disponibilita' | Accesso alle risorse per utenti autorizzati quando serve. | [[CIA_Triad_DAD]] |
| Rischio | Combinazione di vulnerabilita', minacce e valori. | [[Analisi_Rischio]] |
| Plaintext | Messaggio in chiaro. | [[Schemi_di_Cifratura]] |
| Ciphertext | Messaggio cifrato. | [[Schemi_di_Cifratura]] |
| Hash | Funzione one-way che produce digest. | [[Hashing_Message_Digest]] |
| Message digest | Riassunto crittografico a dimensione fissa. | [[Hashing_Message_Digest]] |
| Firma digitale | Firma del digest con chiave privata. | [[Firme_Digitali_Message_Digest]] |
| Certificato X.509 | Associazione firmata tra identita' e chiave pubblica. | [[Certificati_X509_PKI]] |
| Autenticazione | Verifica dell'identita'. | [[Autenticazione]] |
| Controllo accessi | Decisione su operazioni consentite. | [[Controllo_Accessi_SO]] |
| ACL | Diritti associati a un oggetto. | [[ACL_Mediazione_Completa]] |
| DAC | Access control a discrezione del proprietario. | [[DAC_MAC]] |
| MAC | Policy imposta dal sistema. | [[DAC_MAC]] |
| Bell-LaPadula | Modello MAC per confidenzialita'. | [[Bell_LaPadula_Compartimenti]] |
| Biba | Modello MAC per integrita'. | [[Biba]] |
| Packet filter | Firewall basato su header e regole. | [[Packet_Filtering]] |
| DMZ | Rete separata per servizi esposti. | [[Application_Gateway_DMZ]] |
| TLS | Protezione dei dati applicativi sopra TCP. | [[TLS_Struttura_Pacchetto]] |
| IPsec | Protezione a livello IP. | [[IPsec_Modalita_Trasporto_Tunnel]] |
| Weakness | Tipo di errore che puo' introdurre vulnerabilita'. | [[Glossario_Sicurezza_Applicazioni]] |
| Vulnerabilita' | Occorrenza concreta sfruttabile. | [[Glossario_Sicurezza_Applicazioni]] |
| Exploit | Tecnica concreta di sfruttamento. | [[Glossario_Sicurezza_Applicazioni]] |
| XSS | Input interpretato come script dal browser. | [[Cross_Site_Scripting_XSS]] |
| SQL injection | Input che modifica una query SQL. | [[SQL_Injection]] |
| Buffer overflow | Scrittura fuori dai limiti del buffer. | [[Buffer_Overflow]] |
| Buffer overread | Lettura fuori dai limiti del buffer. | [[Buffer_Overread_Heartbleed]] |
