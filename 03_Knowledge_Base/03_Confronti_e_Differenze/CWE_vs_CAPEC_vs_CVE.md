# CWE vs CAPEC vs CVE

| Sigla | Descrive | Livello |
|---|---|---|
| CWE | Weakness | Categoria di difetto |
| CAPEC | Attack pattern | Schema di sfruttamento |
| CVE | Vulnerabilita' concreta | Bug noto in software specifico |

## Da ricordare

CWE dice che tipo di errore evitare. CAPEC dice come un attaccante puo' sfruttare quel tipo di errore. CVE nomina una vulnerabilita' reale, con software e versione.
